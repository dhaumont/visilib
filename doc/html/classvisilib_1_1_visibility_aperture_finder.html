Detailed Description

template<class P, class S>
class visilib::VisibilityApertureFinder< P, S >

Exact visibility determination algorithm based on an aperture finder heuristic. The algorithm stops as soon as an aperture was detected.

This class manage the CSG operations sheduling in Plucker space. It implements an algorithm that searches if at least on aperture exist between the two source polygons, such that at least one line that is intersecting the two source polygons is not blocked by any occluder.
